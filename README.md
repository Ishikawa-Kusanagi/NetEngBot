Установите Python 3.
Установите PostgreSQL и создайте базу данных.
Клонируйте репозиторий с ботом с GitHub.
Установите зависимости:
bash
Копировать код
```
pip install -r requirements.txt
```
Настройте переменные окружения для подключения к базе данных и Telegram API.
Запустите бота:
bash
Копировать код
python main.py
Команды бота
/start - Запуск бота и регистрация пользователя.
/addword - Добавить новое слово с переводом.
/deleteword - Удалить слово.
/quiz - Начать викторину по переводам.
